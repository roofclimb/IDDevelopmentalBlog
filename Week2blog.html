Semantic Web

  • can use other tags instead of div
  • header is used at the top of the page content
  • article is commonly used for news content, blog posts and independent content piece
  • nav is for menu links
  • aside is for content that is indirectly related to your main content

CSS

  • link element: link rel="stylesheet" href="basic.css" type="type/css"
  • can add font colour or words colour colour: red
  • font family and text-align can also be specified
  • ID attribute: #myElement
  • Class attribute: .myClass
  • Pseudo classes
    1. a:link (unvisited)
    2. a:visited (visited)
    3. a:hover (mouse over)
    4. a:active (current link)
    5. a:focus (focus link)
  • Google font can be taken online to use in website
