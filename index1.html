HTML

  • html-defines root of HTML document
  • head-contains information for the document
  • body-includes important information
  • h1 to h6- different headings from biggest to smallest font size
  • p-paragraph
  • ul,ol - unordered list and order list
  • li-defines each list element
  • img-contains an image
  • a-used for hyperlinks
  • br-break
  • hr-horizontal rule
  • table-creates a table
    • th-table heading
    • td-table heading
  • iframe-to embed YouTube videos
  • div-define a section
  • strong/em-bold the text

CSS styling

  1. background color: color
  2. color: red
  3. border size, border color, border-collapse(for border to become one line)
